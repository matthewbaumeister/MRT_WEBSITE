## How to Download Images from Wix

1. **Log into your Wix account**
2. **Access Media Manager**:
   - Go to your Wix dashboard
   - Click on "Media" or "Site Media"
   
3. **Download images**:
   - Find each image listed above
   - Right-click and "Download" or use the download button
   - Save with the filenames listed above

4. **Alternative method** (if you can't access Media Manager):
   - Visit your live Wix site
   - Right-click on each image
   - "Open image in new tab"
   - Right-click and "Save image as..."
   - Note: Quality may be lower with this method

## Image Optimization Tips

Before uploading to the website:

1. **Resize images** to appropriate dimensions (don't use unnecessarily large files)
2. **Compress images** using tools like:
   - TinyPNG (https://tinypng.com)
   - ImageOptim (Mac)
   - Squoosh (https://squoosh.app)
3. **Convert to modern formats** when possible:
   - WebP for better compression (Next.js handles this automatically)
4. **Keep file sizes under**:
   - Hero images: < 500KB
   - Service images: < 200KB
   - Team photos: < 100KB
